# Print Function #

Petal provides a simple print function to output values to the console.

## Examples ##

The basic builtin function is `print(value)`

```
    print(123)
```

This function takes a single value and prints it to the console.

Examples:

```
print(42)        // Prints: 42
print(:hello)    // Prints: :hello
print("world")   // Prints: world
```
