# Instructions

In this directory create a full & working implementation of the Petal programming language.
Use Rust for the implementation.

Read all docs in ../../docs/* to see the technical outline.

The language syntax is not defined yet - please define & implement a syntax that fits the goals.

The working version should have a collection of sample `.ptl` script files that successfully run
using a `petal` binary. All the major features should be demonstrated and working.

When finished provide:
 - Summary of progress & completion status
 - Pain points and challenges that you ran into
 - Feedback on the docs - areas where the docs need improvement?
 - Thoughts for next steps
